# Biblioteca

Sistema de gestão de biblioteca desenvolvido em **Laravel 12**, no âmbito do estágio na **Inovcorp**.

O projeto utiliza **Laravel Jetstream** para autenticação, com **Two-Factor Authentication (2FA)**, e **DaisyUI** para a camada visual baseada em Tailwind CSS.

## Tecnologias Utilizadas

- Laravel 12
- Laravel Jetstream
- Laravel Fortify
- Two-Factor Authentication (2FA)
- Tailwind CSS
- DaisyUI
- MySQL
- Vite

## Funcionalidades Implementadas

- Autenticação de utilizadores
- Registo e login
- Autenticação em dois fatores (2FA)
- Gestão de sessões
- Layout base com DaisyUI

## Funcionalidades em Desenvolvimento

- Gestão de Editoras
- Gestão de Autores
- Gestão de Livros
- Pesquisa, filtragem e ordenação de dados
- Exportação de dados para Excel

## Projeto

Este projeto está a ser desenvolvido de forma incremental, seguindo boas práticas de desenvolvimento web e versionamento com Git.

---
Projeto desenvolvido no contexto de estágio na **Inovcorp**.
